# FreeSurfaceEvaporationRate
This repository contains apps that compute free surface evaporation rate of liquids under the effect of radiation and convection heat fluxes. 

Version 1: Based on the article : Beji, Tarek, and Bart Merci. "Development of a numerical model for liquid pool evaporation." Fire Safety Journal 102 (2018): 48-58. https://doi.org/10.1016/j.firesaf.2018.11.002

Version 2: Modified some changes. Based on the article : Beji, Tarek, and Bart Merci. "Development of a numerical model for liquid pool evaporation." Fire Safety Journal 102 (2018): 48-58. https://doi.org/10.1016/j.firesaf.2018.11.002
